# Financial-Dashboard-VBA
Um dashboard financeiro avançado em Excel com automação VBA para análise de finanças pessoais/empresariais. Inclui modelagem preditiva de fluxo de caixa, análise de risco Monte Carlo, importação automática de dados (de CSV ou API), gráficos interativos e relatórios gerados sob demanda. Ideal para planejamento financeiro, com foco em eficiência…
